HTML - Standardni jezik za označavanje web stranice

OSNOVE:

Pomoću HTML-a može se kreirati sopstvena web lokacija

Šta je HTML?

HTML je skraćenica od Hiper Tektst Markup Language,
HTML je standardni jezik za označavanje za kreiranje veb stranica,
HTML opisuje strukturu veb stranice,
HTML se sastoji od niza elemenata,
HTML elementi govore pretraživaču kako da prikaže sadržaj,
HTML elementi označavaju delove sadržaja kao što su „ovo je naslov“, „ovo je pasus“, „ovo je veza“ itd.


Deklaracija !DOCTYPE html

Deklaracija !DOCTYPE html definiše da je ovaj dokument HTML5 dokument,
Element html je osnovni element HTML stranice,
Element head sadrži meta informacije o HTML stranici,
Element title navodi naslov za HTML stranicu (koji je prikazan u naslovnoj traci pretraživača ili na kartici stranice),
Element bodi definiše telo dokumenta i predstavlja kontejner za sav vidljivi sadržaj, kao što su naslovi,
pasusi, slike, hiperveze, tabele, liste itd.,
Element h1 definiše veliki naslov,
Element p definiše pasus.

Šta je HTML element?

HTML element je definisan početnom oznakom, nekim sadržajem i završnom oznakom.

Veb pretraživači

Svrha veb pretraživača (Chrome, Edge, Firefox, Safari) je da čita HTML dokumente i da ih ispravno prikaže.
Pregledač ne prikazuje HTML oznake, ali ih koristi da odredi kako da prikaže dokument.

Svi HTML elementi mogu imati atribute

Href atribut a specificira URL stranice na koju link vodi,
Src atribut img određuje putanju do slike koja će biti prikazana,
Atributi širine i visine img pružaju informacije o veličini slika,
Atribut alt img obezbeđuje alternativni tekst za sliku,
Atribut style se koristi za dodavanje stilova elementu, kao što su boja, font, veličina i još mnogo toga,
Atribut lang oznake html deklariše jezik veb stranice,
Atribut title definiše neke dodatne informacije o elementu.
